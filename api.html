API reference#

wce_editor.js#

setWceEditor(_id[, clientOptions, baseURL, callback])#

Initialises the editor

Arguments:

  • _id (string) – The html id value of the text area to transform into the editor.

  • clientOptions (clientOptions) – The client settings to use when initialising the editor.

  • clientOptions.language (string) – The language to use for the interface (en|de). The default is English.

  • clientOptions.rtl (boolean) – This should be set to true for transcribing right to left languages. The default is false.

  • clientOptions.getWitness (function) – An optional function to get the value of the current witness.

  • clientOptions.getWitnessLang (function) – An optional function to get the language of the current witness.

  • clientOptions.getBookNameFromBKV (function) – A function to switch BKV references to OSIS (only used for book and chapter). This function is provided with two arguments: the B or BK reference depending on the context (i.e. B04, B04K1); the full n attribute value of the first ab following the Book or Chapter.

  • clientOptions.bookNames (array) – A list of OSIS book abbreviations to use in the select of the V menu. If this list is not supplied the form will have a text box for manual entry.

  • clientOptions.addLineBreaks (boolean) – Add line breaks in the XML before every pb, cb and lb in the transcription. Default is false.

  • clientOptions.addSpaces (boolean) – Add spaces into the XML of the transcription between tags to make the text readable if all the tags are removed. Default is false.

  • clientOptions.defaultReasonForUnclearText (string) – The default option in the reason box for unclear text. Default pre-selects nothing.

  • clientOptions.defaultHeightForCapitals (number) – If a number is supplied with this settings then it is used to prepopulate the height box in the O > capitals submenu. If it is not supplied then the box is not prepopulated.

  • clientOptions.defaultValuesForSpaceMenu (defaultValuesForSpaceMenu) – The default settings to use to pre-poulate the spaces menu. Default does not pre-populate anything but does select first option on the unit list.

  • clientOptions.defaultValuesForSpaceMenu.unit (string) – The unit value to preselect.

  • clientOptions.defaultValuesForSpaceMenu.extent (number) – The extent number to preselect.

  • clientOptions.showMultilineNotesAsSingleEntry (boolean) – If set to true this combines multiline untranscribed commentary and lectionary notes into a single line (does not change the XML output). Default is false.

  • clientOptions.checkOverlineForAbbr (boolean) – If set to true this will check the ‘add overline’ option in the abbreviation form when it is loaded. Default is false.

  • clientOptions.optionsForGapMenu (optionsForGapMenu) – The options used to create and to set defaults in the gap menu.

  • clientOptions.optionsForGapMenu.reason (string) – The option to select by default for the reason for the gap (illegible|lacuna|unspecified|inferredPage).

  • clientOptions.optionsForGapMenu.suppliedSource (string) – The option to use for the source of the supplied text.

  • clientOptions.optionsForGapMenu.sourceOptions (Array.<sourceOptions>) – An optional list of sources to use for the supplied source dropdown. None and other are always present and cannot be changed by this setting the remaining default are most relevant to Greek New Testament.

  • clientOptions.optionsForGapMenu.sourceOptions.value (string) – The value to record in the XML for this supplied source.

  • clientOptions.optionsForGapMenu.sourceOptions.labelEn (string) – The visible label to use for this entry in the English interface.

  • clientOptions.optionsForGapMenu.sourceOptions.labelDe (string) – The visible label to use for this entry in the German interface.

  • clientOptions.optionsForMarginaliaMenu (optionsForMarginaliaMenu) – The options used to create and to set defaults in the marginalia menu.

  • clientOptions.optionsForMarginaliaMenu.type (string) – The option to select by default for the fw_type dropdown for the marginalia.

  • clientOptions.transcriptionLanguage (string) – The css to use for the transcription in the editor. Choices are currently coptic and greek. Default is greek.

  • clientoptions.showLineNumberSidebarOnLoading (boolean) – A boolean to determine if the line number sidebar should be shown on loading or not. Default is True

  • clientOptions.toolbar (string) – The string to use to configure the toolbar. It should be a subset of the default provided, | puts a divider at that point in the toolbar.

  • clientOptions.preferredFont (string) – The preferred font-family string to use for the transcription in the editor.

  • string (baseURL) – Explicitly sets TinyMCE’s base URL.

  • function (callback) – The function to call once the editor is loaded.

setTEI(teiStringInput)#

Set the content of the editor from TEI-XML input. This can be used as the callback to setWceEditor to preload a basetext or a partially made transcription.

Arguments:

  • teiStringInput (String) – The TEI-XML as a string to display in the editor.

getTEI()#

Get TEI-XML string from editor html content.
