    Name Type Description
    language string The language to use for the interface (en|de). The default is English.
    rtl boolean This should be set to true for transcribing right to left languages. The default is false.
    getWitness function An optional function to get the value of the current witness.

    addSpaces boolean Add spaces into the XML of the transcription between tags to make the text readable if all the tags are removed. Default is false.
